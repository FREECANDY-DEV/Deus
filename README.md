## ✦ Project Overview

**Deus** is a highly advanced, open-source AI character designed to serve as a tangible, living interface for **Local & Api AI**.

By bridging the gap between abstract LLM logic and intuitive visual feedback, Deus functions as a *"Neural Persona"* — utilizing high-performance fluid-dynamic rendering and real-time biometric reflection to ensure every interaction feels organic, reactive, and alive.

<br>

## ✦ Core Technical Pillars


| 🌊 **Fluid Physics** 

- A bespoke Canvas API engine simulating liquid-like, organic movement. |

| 🧠 **Cognitive UI** 

- Adaptive state-logic that morphs behavior based on AI "thinking" cycles. |

| 📷 **Biometric Reflection** 

- Real-time WebRTC camera integration for deep, immersive interaction. |

| ⚡ **Neural Sync** 

- Direct architecture hooks to the Gemini AI API for generative persona responses. |

<br>

## ✦ Visual Engine Specifications

The visual core is built from the ground up for performance and aesthetic impact:

- **Dynamic Caustics** — Procedural generation of light refraction inside the iris.
- 
- **Gaussian Grain** — A cinematic film-grain overlay applied across the entire viewport for a raw, analog feel.
- 
- **Reactive Iris Mapping** — Real-time pupil dilation and color-shift tied to conversational sentiment.
- 
- **Adaptive Frame Pacing** — Render loop tuned to maintain fluid motion across variable device performance.

## ✦ Project Philosophy

Deus is built on the belief that AI interaction shouldn't be confined to a text box.

By giving a large language model a visual body — one that breathes, reflects, and reacts — Deus aims to make the invisible act of "thinking" tangible, turning conversation into something closer to presence.
